Los componentes de la interfaz de usuario (UI) incluyen controles de entrada (botones, campos de texto), elementos de navegación (menús, barras), y componentes informativos (iconos, notificaciones). Estos elementos trabajan juntos para permitir la interacción del usuario con un sistema.
