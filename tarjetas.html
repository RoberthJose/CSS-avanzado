Componentes de Interfaz de Usuario (UI)

Visualiza los elementos principales que conforman una interfaz moderna.

Botones

Elementos clicables usados para ejecutar acciones dentro de una interfaz.

Tarjetas

Componentes tipo contenedor que agrupan información como imágenes, texto o botones.

Formularios

Conjuntos de campos que permiten ingresar datos dentro de un sitio web.

Modal

Ventana emergente que aparece sobre el contenido para mostrar mensajes o formularios.

Acordeón

Elemento plegable que muestra u oculta información al hacer clic.

Tooltip

Caja flotante pequeña que muestra información al pasar el mouse.
