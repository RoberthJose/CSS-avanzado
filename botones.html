Componentes de UI: Botones

Explora los botones y sus animaciones.

Botón Primario

Se utiliza para acciones principales o muy importantes.

Botón Secundario

Se utiliza para acciones secundarias o menos importantes.

Botón Éxito

Se usa para indicar acciones exitosas o confirmar procesos.

Botón Peligro

Indica advertencias o acciones que pueden eliminar datos.

Botón Advertencia

Se usa para alertar sobre situaciones muy importantes.

Botón de Link

Este botón sirve para redirigir a otra página dentro del sitio.
